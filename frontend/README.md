# Nexora ERP Frontend

High-performance ERP interface built with React, Vite, and Tailwind CSS.

## Features
- **Hexagonal Dashboard**: Axial coordinate-based mesh layout.
- **Role-Based Access**: Granular permission management via `AccessLevelJSON`.
- **Session Integrity**: Tab-fingerprinting and multi-session conflict detection.
- **Scrambled Payloads**: XOR-based bidirectional data security.

## Tech Stack
- **Framework**: React 19
- **Build Tool**: Vite 7
- **Styling**: Tailwind CSS 4
- **Auth**: Firebase Authentication
- **Database**: Firebase Firestore & FileMaker Data API

## Development
```bash
npm install
npm run dev
```
